## `dawn.py`

> A simpler version of `owo-dusk` with input system.

### Controls

| Key |  Command | | Key |  Command |
| :---: | :--- | :---: | :---: | :--- |
| **`1`** | Toggle Battle  | | **`8`** | Open Lootboxes |
| **`2`** | Toggle Hunt    | | **`R`** | Watchdog Unpause |
| **`3`** | Toggle OwO     | | **`W`** | Toggle Open Website |
| **`4`** | Toggle Pray    | | **`=`** | + 0.25 Cooldown |
| **`5`** | Toggle Curse   | | **`-`** | - 0.25 Cooldown |
| <img src=https://capricho.abril.com.br/wp-content/uploads/2026/04/67-kid.jpg width=20> | Toggle Open Crates  | | **`C`** | Toggle Captcha |
| <img src=https://capricho.abril.com.br/wp-content/uploads/2026/04/67-kid.jpg width=20> | Open Boss Crates    | | **`X`** | Exit |
